# 📊 Top US Companies Dashboard

A data analytics project that demonstrates the complete workflow of collecting, processing, and visualizing data. The project uses Python to scrape information about top companies in the United States from Wikipedia and presents key insights through an interactive Power BI dashboard.

---

## 📌 Project Overview

This project was created to practice and showcase:

- Web Scraping with Python
- Data Cleaning and Preparation
- Business Intelligence Reporting
- Dashboard Development using Power BI

The dataset was extracted from Wikipedia, processed using Python, and visualized through an interactive dashboard.

## 🚀 Project Workflow

### 1. Data Collection
- Scraped data from Wikipedia using Python.
- Extracted information about leading companies in the United States.

### 2. Data Cleaning
- Removed inconsistencies and missing values.
- Structured the dataset for analysis and reporting.

### 3. Dashboard Development
- Imported the cleaned dataset into Power BI.
- Created interactive visualizations and reports.
- Built a dashboard to provide quick insights into the data.
